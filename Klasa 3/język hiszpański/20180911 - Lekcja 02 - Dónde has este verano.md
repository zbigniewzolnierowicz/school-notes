## Czas przeszły pasado reciente

haber - mieć

| osoba              | czasownik haber |
| ------------------ | --------------- |
| yo                 | hé              |
| tú                 | has             |
| el, ella, Vd.      | ha              |
| nos                | hemos           |
| vos                | habéis          |
| ellas, ellos, Vds. | han             |

`-er/ir => ido`

ver => visto - widzieć

poner => puesto - nosić
